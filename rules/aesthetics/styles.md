# 67 UI Styles Reference

## General Styles

### 1. Minimalism & Swiss Style
Grilles strictes, typographie clean, beaucoup d'espace blanc. Pas de decoration inutile.

### 2. Neumorphism
Ombres douces internes/externes qui simulent l'extrusion. Fond clair uniforme.

### 3. Glassmorphism
Effet verre depoli. backdrop-filter: blur, fond semi-transparent, bordure subtile.

### 4. Brutalism
Brut, non-poli. Bordures epaisses, couleurs vives, typo large, anti-convention.

### 5. Neubrutalism
Brutalism adouci. Ombres portees hard, bordures noires, couleurs pastels vives.

### 6. Flat Design
Zero ombres, zero gradients. Couleurs solides, icones simples, typographie clean.

### 7. Dark Mode (OLED)
Fond noir pur (#000), accents vifs, contraste maximal. Ideal pour OLED.

### 8. Bento Box Grid
Grille irreguliere style Apple. Cards de tailles differentes, asymetrie controlee.

### 9. Retro-Futurism
Melange retro 80s et futuriste. Neons, grilles, couleurs saturees sur fond sombre.

### 10. Cyberpunk UI
Neons, glitch effects, HUD elements, tech dystopique.

### 11. Y2K Aesthetic
Bulle, chrome, couleurs irisees, formes organiques, nostalgie 2000s.

### 12. Vaporwave
Pastels, gradients, esthetique internet 90s, grilles, statues grecques.

### 13. Memphis Design
Geometric, colorful, patterns, anti-minimaliste, playful.

### 14. Aurora UI
Gradients aurora boreale, fonds sombres, couleurs etherees.

### 15. Motion-Driven
L'animation est le design. Scroll-triggered, transitions fluides, micro-interactions.

### 16. Spatial UI (VisionOS)
Style Apple Vision Pro. Verre, profondeur, layers, materials.

### 17. E-Ink / Paper
Aspect papier/e-ink. Faible contraste, textures papier, typographie editoriale.

### 18. Gen Z Chaos / Maximalism
Tout a la fois. Couleurs clashantes, typo mixte, collage, energy.

### 19. Organic Biophilic
Formes naturelles, couleurs terre, textures organiques, courbes douces.

### 20. Liquid Glass
Surfaces liquides, reflets, distorsions, transparences dynamiques.

### 21. Soft UI Evolution
Neumorphism evolue. Plus subtil, meilleur contraste, plus accessible.

### 22. Dimensional Layering
Couches de profondeur. Z-axis evident, parallaxe, shadows multiples.

### 23. Exaggerated Minimalism
Minimalisme pousse a l'extreme. Une couleur, une font, maximum d'espace.

### 24. Kinetic Typography
La typographie bouge. Animations de texte comme element de design principal.

### 25. Parallax Storytelling
Narration par le scroll. Couches a vitesses differentes.

### 26. Swiss Modernism 2.0
Swiss style modernise. Grilles, typo, mais avec des couleurs et du mouvement.

### 27. HUD / Sci-Fi FUI
Interface a la Minority Report. Grilles, cercles, data visualization futuriste.

### 28. Pixel Art
Retro gaming. Pixels visibles, palette limitee, nostalgie 8-bit.

### 29. Editorial Grid / Magazine
Layout magazine. Colonnes, pull quotes, images full-bleed, typo editoriale.

### 30. Gradient Mesh / Aurora Evolved
Gradients mesh complexes. Multi-couleurs, organiques, immersifs.

### 31. Anti-Polish / Raw Aesthetic
Intentionnellement brut. Pas de polish, authentique, underground.

### 32. Tactile Digital / Deformable UI
Elements qui reagissent au touch. Deformation, elasticite, physique.

### 33. Nature Distilled
Nature abstraite en design. Couleurs nature, formes organiques simplifiees.

### 34. Interactive Cursor Design
Le curseur est un element de design. Custom cursors, trails, interactions.

### 35. 3D Product Preview
Produits en 3D interactif. Rotation, zoom, configuration.

### 36. Chromatic Aberration / RGB Split
Effet aberration chromatique. Decalage RGB, glitch subtil, cyber.

### 37. Vintage Analog / Retro Film
Grain film, couleurs desaturees, typographie retro, nostalgie analogique.

### 38-49. Styles supplementaires
Skeuomorphism, Claymorphism, 3D & Hyperrealism, Vibrant & Block-based, Accessible & Ethical, Micro-interactions focus, Inclusive Design, Zero Interface, Voice-First Multimodal, Interactive Cursor Design, 3D Product Preview, Chromatic Aberration.

## Landing Page Styles

### 50. Hero-Centric
Hero massif plein ecran, scroll reveal pour le contenu.

### 51. Conversion-Optimized
Chaque element oriente vers la conversion. CTA visible, friction minimale.

### 52. Feature-Rich Showcase
Chaque feature a sa section avec demo/screenshot.

### 53. Minimal & Direct
Message direct, peu de sections, CTA immediat.

### 54. Social Proof-Focused
Temoignages et logos = le coeur de la page.

### 55. Interactive Product Demo
Demo interactive en hero, l'utilisateur teste avant de s'inscrire.

### 56. Trust & Authority
Certifications, logos, etudes de cas = credibilite.

### 57. Storytelling-Driven
Narration progressive, hero → probleme → solution → CTA.

## Dashboard Styles

### 58. Data-Dense
Maximum d'info, grilles serrees, pour power users.

### 59. Executive Dashboard
KPIs clairs, peu de data, vue high-level pour decideurs.

### 60. Real-Time Monitoring
Donnees live, updates automatiques, alertes visuelles.

### 61. Drill-Down Analytics
Vue globale → clic → detail. Navigation par les donnees.

### 62. Comparative Analysis
Side-by-side, benchmarks, deltas, trends.

### 63. Predictive Analytics
Projections, forecasts, confidence intervals, ML insights.

### 64. User Behavior Analytics
Funnels, heatmaps, parcours utilisateur, retention curves.

### 65. Financial Dashboard
P&L, cash flow, burn rate, runway. Precision chiffree.

### 66. Sales Intelligence
Pipeline, deals, forecasts, performance par rep.

### 67. Heat Map Style
Couleurs pour representer l'intensite. Calendriers, matrices.
