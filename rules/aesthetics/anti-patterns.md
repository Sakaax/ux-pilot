# Anti-Patterns "AI Slop"

## Fonts a EVITER
- Inter, Roboto, Arial, system fonts — signalent un manque d'intentionalite
- Open Sans, Lato — trop generiques, zero personnalite
- Utiliser une seule police partout sans hierarchie visuelle

## Fonts a PRIVILEGIER
- Code aesthetic : JetBrains Mono, Fira Code, Space Grotesk
- Editorial : Playfair Display, Crimson Pro, Fraunces
- Startup : Clash Display, Satoshi, Cabinet Grotesk
- Technique : IBM Plex family, Source Sans 3
- Distinctif : Bricolage Grotesque, Obviously, Newsreader

## Couleurs a EVITER
- Purple gradients on white backgrounds — cliche "AI-generated"
- Palettes timides reparties uniformement sans accent dominant
- Bleu generique (#0066FF) sans raison

## Couleurs a PRIVILEGIER
- Une couleur dominante + accent sharp > palette timide
- Palette alignee avec le produit/industrie
- Dark mode : desaturer, pas juste inverser
- Contraste intentionnel (pas uniformement moyen)

## Layouts a EVITER
- Meme structure cookie-cutter sur toutes les pages
- Grilles previsibles sans variation
- Hero section generique (titre centre + sous-titre + bouton)

## Layouts a PRIVILEGIER
- Asymetrie intentionnelle
- Overlap d'elements
- Negative space genereux OU densite controlee
- Grid-breaking elements pour casser la monotonie

## Backgrounds a EVITER
- Couleur unie plate sans profondeur
- Blanc pur (#FFFFFF) partout sans texture ni atmosphere

## Backgrounds a PRIVILEGIER
- Gradient meshes subtils
- Noise textures
- Geometric patterns
- Layered transparencies
- Shadows dramatiques pour la profondeur
- Grain overlays pour le caractere

## Typographie craft
- Weight extremes : 100/200 vs 800/900 (pas 400 vs 600)
- Size jumps de 3x+ entre heading et body
- Display font + mono/serif pairing > deux sans-serif similaires
- Charger depuis Google Fonts, declarer le choix avant de coder
