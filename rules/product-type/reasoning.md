# Product-Type Reasoning Rules

Recommandations specifiques par type de produit. Chaque entree definit : landing pattern, styles recommandes, focus couleur, style dashboard, et anti-patterns a eviter.

## SaaS (General)
- Landing: Hero + Features + CTA
- Styles: Minimalism, Flat Design
- Colors: Bleu primaire, accents vifs
- Dashboard: Data-Dense ou Executive
- Avoid: trop de fun, manque de serieux

## Micro SaaS
- Landing: Minimal & Direct
- Styles: Exaggerated Minimalism
- Colors: Mono ou duo-tone
- Avoid: complexite inutile, trop de features affichees

## Marketplace
- Landing: Marketplace/Directory
- Styles: Bento Box Grid
- Colors: Neutre avec accents pour les categories
- Dashboard: User Behavior Analytics
- Avoid: design trop charge, navigation confuse

## E-commerce
- Landing: Product Demo + Social Proof
- Styles: Clean, photos produit en hero
- Colors: Neutre avec accent pour les CTA
- Dashboard: Sales Intelligence
- Avoid: texte trop dense, images basse qualite

## Fintech / Crypto
- Landing: Trust & Authority + Conversion
- Styles: Dark Mode, Minimal
- Colors: Or/gold accents, fonds sombres
- Dashboard: Financial Dashboard
- Avoid: couleurs playful, manque de securite percue

## Developer Tool
- Landing: Feature-Rich Showcase ou FAQ/Doc
- Styles: Dark Mode, Code aesthetic
- Colors: Sombre, accents neon (vert, bleu, violet)
- Dashboard: Real-Time Monitoring
- Avoid: marketing fluff, manque de doc technique

## AI / Chatbot Platform
- Landing: Interactive Product Demo
- Styles: Aurora UI, Gradient Mesh
- Colors: Violet, bleu profond, accents lumineux
- Dashboard: Predictive Analytics
- Avoid: purple gradients generiques (AI slop), manque de substance

## Healthcare / Medical
- Landing: Trust & Authority
- Styles: Clean, Accessible, Soft
- Colors: Bleu calme, vert, blanc abondant
- Dashboard: Executive Dashboard
- Avoid: design trop bold, couleurs anxiogenes (rouge dominant)

## Education / EdTech
- Landing: Hero + Demo + Social Proof
- Styles: Fun/Playful, Colorful
- Colors: Vifs mais pas agressifs, palettes chaudes
- Dashboard: User Behavior Analytics
- Avoid: trop corporate, trop dense

## Creative Agency
- Landing: Portfolio Grid ou Immersive
- Styles: Bold, Motion-Driven, Editorial Grid
- Colors: Noir + accent fort (rose, orange, neon)
- Avoid: templates generiques, manque de personnalite

## Restaurant / Food
- Landing: Hero-Centric avec photo food
- Styles: Warm, Organic
- Colors: Rouge, or, terre, accents chauds
- Avoid: design froid, manque de sensorialite

## Fitness / Health
- Landing: Before-After Transformation
- Styles: Bold, Energetique
- Colors: Neon vert/orange sur fond sombre
- Dashboard: Progress tracking
- Avoid: design trop calme, manque d'energie

## Real Estate
- Landing: Search + Listings
- Styles: Premium, Clean
- Colors: Bleu fonce, or, blanc
- Avoid: design trop tech, manque de confiance

## Travel / Tourism
- Landing: Immersive, Photo-driven
- Styles: Parallax Storytelling
- Colors: Bleu ciel, orange sunset, vert nature
- Avoid: trop de texte, manque d'images immersives

## Legal / Consulting
- Landing: Trust & Authority
- Styles: Conservative, Editorial
- Colors: Bleu marine, gris, or discret
- Avoid: design trop moderne/disruptif, manque de gravite

## Gaming
- Landing: Immersive, 3D, Video hero
- Styles: Cyberpunk, HUD/Sci-Fi, Dark
- Colors: Neon, saturees, fond sombre
- Avoid: design corporate, manque d'immersion

## Music / Entertainment
- Landing: Hero-Centric, Video-First
- Styles: Dark, Bold, Motion-Driven
- Colors: Contrastes forts, accents neon
- Avoid: design generique, grilles rigides

## Productivity Tool
- Landing: Feature-Rich Showcase
- Styles: Minimal, Clean
- Colors: Neutre avec un accent distinctif
- Dashboard: Data-Dense
- Avoid: trop de decoration, manque de focus fonctionnel

## Social Platform
- Landing: Community/Forum
- Styles: Fun, Colorful, Bento
- Colors: Violet/rose/bleu vif (mais pas generique)
- Dashboard: User Behavior Analytics
- Avoid: design trop corporate

## Luxury / Fashion
- Landing: Hero-Centric, Editorial Grid
- Styles: Premium, Exaggerated Minimalism
- Colors: Noir, or, blanc casse, accents discrets
- Avoid: design cheap, couleurs vives, surcharge d'info

## B2B Enterprise
- Landing: Enterprise Gateway
- Styles: Corporate, Trust-focused
- Colors: Bleu, gris, blanc, accents discrets
- Dashboard: Executive ou Comparative Analysis
- Avoid: design trop fun, manque de serieux

## Open Source Tool
- Landing: FAQ/Doc + Feature Showcase
- Styles: Developer aesthetic, Dark Mode
- Colors: Fond sombre, accents vifs
- Dashboard: Real-Time Monitoring
- Avoid: marketing fluff, paywalls en avant

## Newsletter / Media
- Landing: Content First
- Styles: Editorial Grid, Typographie forte
- Colors: Noir + blanc + un accent
- Avoid: trop d'images, manque de focus contenu

## Booking / Appointment
- Landing: Hero + Search + Calendar
- Styles: Clean, Functional
- Colors: Bleu calme, vert pour disponibilite
- Avoid: design trop decoratif, processus complexe

## HR / Recruitment
- Landing: Trust + Social Proof
- Styles: Professional, Accessible
- Colors: Bleu, violet doux
- Dashboard: Comparative Analysis
- Avoid: design trop fun pour le B2B, trop froid pour le B2C
